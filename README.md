# Plataforma Enel

Projeto web estático para trilha de aprendizagem gamificada.

## Executar localmente

```bash
python3 -m http.server 5500
```

Acesse: `http://localhost:5500`

## Observações

- Login de teste: `usuário` / `123`
- Durante testes, o quiz pode ser iniciado sem concluir o vídeo.
- Para usar os vídeos de fase em produção, adicione os arquivos:
  - `assets/enel/videos/fase-1.mp4`
  - `assets/enel/videos/fase-2.mp4`
  - `assets/enel/videos/fase-3.mp4`
- A fonte Roobert Enel pode ser colocada em `assets/enel/fonts/` conforme instruções do README da pasta.
