# Fontes Enel (Roobert Enel)

Coloque aqui os arquivos oficiais da fonte Roobert Enel.

## Nomes esperados pelo projeto
- `RoobertEnel-Regular.woff2`
- `RoobertEnel-Regular.woff`
- `RoobertEnel-Bold.woff2`
- `RoobertEnel-Bold.woff`

Se receber em OTF/TTF, converta para WOFF/WOFF2 mantendo esses nomes.

## Caminho da pasta
`assets/enel/fonts/`
